⚡ QUICK START (Choose your platform):

WINDOWS:
  1. Extract the ZIP file
  2. Navigate into the "fleet-safety-crm" folder
  3. Double-click "start.bat" ← that's it!
  4. Wait ~2-3 minutes for dependencies to install
  5. Your browser opens automatically at http://localhost:5173

MAC/LINUX:
  1. Extract the ZIP file
  2. Open Terminal
  3. Type: cd path/to/fleet-safety-crm
  4. Type: ./start.sh
  5. Wait ~2-3 minutes for dependencies to install
  6. Your browser opens automatically at http://localhost:5173

MANUAL (All platforms):
  1. Extract the ZIP file
  2. Open Terminal/Command Prompt in the folder
  3. Type: npm install
  4. Type: npm run dev
  5. Visit http://localhost:5173
